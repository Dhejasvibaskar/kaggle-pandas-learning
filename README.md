
# Kaggle Pandas Course

Description:
Solutions and practice notebooks from the Kaggle Pandas course. Covers DataFrame creation, reading/writing data, indexing, selection, grouping, sorting, and data analysis using Python and Pandas.

This repository contains my solutions and notes from the Kaggle Pandas course.

## Topics Covered
- Creating, Reading, and Writing Data
- Indexing, Selecting & Assigning
- Summary Functions
- Grouping and Sorting
- Data Types and Missing Values
- Renaming and Combining Data

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Kaggle

## Goal
To build a strong foundation in data manipulation and analysis using Pandas and prepare for real-world data science and analytics projects.

Course: https://www.kaggle.com/learn/pandas
